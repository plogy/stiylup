Plogy

Plogy is a logging library for python developers who want multiple choices to notify their users. It provides email, gui, and text notifications. It also provides style for your Errors, Warnings, and more.

See Github Repo See Documentation

Features

Easy Colors

You are provided a toolkit for styling text which includes in-string styles, json styles, and toml styles.

Email Notifications

You can send your user emails for notifications. 

GUI Notifications

You can make GUI notifications easily to notify your users stylishly.
